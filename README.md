# STARMOVIES 🍿⭐️

Aplicativo para avaliar filmes desenvolvido com SwiftUI + SwiftData

## 🍎 DUPLA:

✦ [Leticia Gomes](https://github.com/mareshbard)
✦ [Yohane Cavalcante](https://github.com/whyohane)

## 🎬 FUNCIONALIDADES:

- Adicionar filme
- Editar
- Excluir
- Listar
- Avaliar
